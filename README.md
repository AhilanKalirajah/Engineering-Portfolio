# Ahilan Kalirajah Portfolio Site

Static GitHub Pages portfolio generated from the attached resume and portfolio PDFs, adapted to the HTML5 UP Future Imperfect theme.

## Publish With GitHub Pages

1. Create a GitHub repository named `your-username.github.io` for a personal site, or any repository name for a project site.
2. Upload everything in this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Choose `main` and `/root`, then save.

GitHub will publish the site after the Pages build finishes.

## Files

- `index.html` - portfolio content
- `styles.css` - responsive page styling
- `script.js` - project filtering
- `assets/Resume.pdf` - downloadable resume
- `assets/Portfolio.pdf` - downloadable full portfolio
- `assets/images/` - all 49 extracted portfolio images
- `LICENSE.txt` - HTML5 UP theme license
- `THEME-README.txt` - original theme readme
