# Ahilan Kalirajah Portfolio Site

Static GitHub Pages portfolio redesigned as a mechanical engineering case-study site.

## Publish With GitHub Pages

1. Upload everything in this folder to the repository root.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose `main` and `/root`, then save.

## Files

- `index.html` - portfolio structure and page content
- `styles.css` - responsive visual design
- `script.js` - project data, filtering, and expandable case studies
- `Portfolio.pdf` - downloadable portfolio PDF
- `images/` - existing project images
